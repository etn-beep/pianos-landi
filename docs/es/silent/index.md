# Sistema Silent

El sistema Silent permite tocar el piano sin sonido acústico gracias a sensores MIDI...

* Puedes:
  tocar a cualquier hora sin molestar a quienes te rodean,
  dar nueva vida a un piano antiguo,
  evitar la necesidad de invertir en un piano digital además de tu piano acústico,
  disfrutar de la auténtica sensación de un piano real con las ventajas del sistema digital,
  trabajar al mismo tiempo en un piano acústico y producir música asistida por ordenador (DAW).

![silent](../images/silent01.jpg)

### Funcionamiento:

* Sensores instalados bajo el teclado transforman cada tecla pulsada en una señal MIDI que se envía a un procesador equipado con una tarjeta de sonido. El sistema no modifica en absoluto la sensación al tacto de un teclado acústico tradicional.

* Una barra de parada de martillos bloquea estos en posición de “silencio” antes de que golpeen las cuerdas. De este modo, el piano deja de producir sonido acústico y el sistema MIDI toma el relevo. Una palanca discreta situada bajo el teclado permite cambiar entre el modo acústico, el modo silencioso o incluso ambos a la vez.

* También puedes grabarte mientras tocas, lo que facilita el trabajo de interpretación y el estudio musical.

* La unidad de control incluye diferentes sonidos y múltiples funciones como metrónomo, pistas de grabación y efectos. Puede conectarse a un ordenador y a otros dispositivos compatibles MIDI, así como a sistemas de amplificación.

* El sistema puede instalarse en **casi todos los pianos**:
  nuevos o de segunda mano, con una instalación sencilla o con modificaciones más o menos importantes según el modelo.

* Existen varias marcas y modelos disponibles.
  Consulta siempre con un profesional especializado.
